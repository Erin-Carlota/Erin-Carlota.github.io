# Spin 加载中 pass

📜 Attributes

参数 说明 类型 可选值 默认值
spinning 是否显示加载效果，默认不显示 Boolean —— false
delay 延迟显示加载效果出现的时间，单位为 ms，在加载效果出现前修改 spinning 为 false，则不出现加载效果（防止闪烁） Number —— ——
width 加载动画的直径大小，单位为 px Number —— 80
tips 加载状态的提示文字，将出现在加载动画的正下方 String —— ——
